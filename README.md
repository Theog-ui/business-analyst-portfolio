# Business Analyst / Content Ops Mini‑Portfolio — Lopamudra Naik

This pack shows scenario/case writing, structured formats (TSV/CSV, JSON, HTML, XML, Markdown),
QA rubrics, SOPs, stakeholder updates, and simple reporting. All data is synthetic and safe to share.

**Files**
- `scenarios.tsv` — MBA‑style mini‑scenarios with inputs, constraints, acceptance criteria, and edge cases.
- `scenarios.xml` — Same scenarios in XML form.
- `scenarios.html` — HTML table version (for browser preview or wiki pastes).
- `qa_rubric.md` — 10‑dimension rubric with weights and scoring anchors.
- `brief.json` — Structured business brief for a pricing/packaging decision.
- `data_dictionary.md` — Column definitions and types for the TSV/JSON.
- `sop_content_ops.md` — Intake → assignment → two‑step QA → delivery process with SLAs and RACI.
- `case_study_mba_style.md` — One‑page case study (before/after metrics).
- `qa_log.tsv` — Sample issue log during reviews.
- `status_update_template.md` — Weekly stakeholder update template.
- `metrics.csv` — Synthetic weekly metrics (throughput, first‑pass approval, turnaround time).
- `metrics_trends.png` — Chart of quality/speed metrics.
- `outreach_note.md` — Short note you can send with an application.
- `pinned_post.md` — Caption to pin on LinkedIn with the portfolio link.
